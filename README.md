This project contains the workflow to build a simple website with recipes on.
Once I have completed this project I will have demonstrated the following skills:

1. Creating a boilerplate html file for the website.
2. Populating the html file with the learned elements and tags, such as headings, titles, paragraphs, unordered/ordered lists, links, and images.
3. Link pages together on the website.
4. Appropriately use Git to make edits, add, commit, and push changes to the repo.
